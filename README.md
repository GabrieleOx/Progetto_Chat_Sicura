# Progetto_Chat_Sicura
Chaìt Sicura client-server con database MariaDB

**Sicurezza:**

Cifratura simmetrica con controllo di integrità: AES-GCM

**Implementazione:**
Python terminale, con multiple finestre per ogni chat point to point
