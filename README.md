# Progetto_Chat_Sicura
Chat Sicura p2p con architettura client-server con database MariaDB <br>

**Sicurezza:** <br>
Cifratura asimmetrica per lo scambio di chiavi simmetriche: RSA 3072 B <br>
Cifratura simmetrica con controllo di integrità: AES-GCM <br>

**Implementazione:** <br>
Python Textual UI, con più canali, uno per ogni chat <br>
